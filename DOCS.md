# BookIt Project Documentation Index

This folder contains detailed documentation for the BookIt platform, a campus food pre-booking platform.

## 📁 Directory Structure

- [**Architecture Overview**](./docs/architecture.md) — System design and data flow.
- [**Backend Documentation**](./docs/backend/README.md) — FastAPI, PostgreSQL, MinIO, and WebSockets.
- [**Frontend Documentation**](./docs/frontend/README.md) — React + Vite for students and canteen owners.
- [**Admin Documentation**](./docs/admin/README.md) — Admin panel for managing owners and canteen approvals.

## 🛠️ Tech Stack

### Backend
- **Framework:** FastAPI
- **Database:** PostgreSQL (SQLAlchemy + asyncpg)
- **Authentication:** JWT (jose)
- **Storage:** MinIO (S3-compatible)
- **Real-time:** WebSockets

### Frontend
- **Framework:** React (Vite)
- **Styling:** CSS Modules / Vanilla CSS
- **State Management:** React Context API
- **Routing:** React Router v6
- **Notifications:** React Hot Toast
